# Multi-Converter App 🛠️📏

A powerful and versatile conversion tool built with **React Native** and **Expo**. This app is designed to simplify daily tasks by gathering everything from real-time currency exchange to kitchen measurements in one place.

## 🚀 Modules

The app is divided into four main categories to cover all your conversion needs:

- **💱 Currency:** Real-time conversion between 30+ global currencies using the Frankfurter API.
- **⚖️ Weight:** Seamlessly switch between grams, kilograms, pounds, and ounces.
- **📏 Length:** Convert everything from millimeters to miles.
- **🍞 Baking & Volume:** Perfect for the kitchen! Convert between liters, deciliters, tablespoons, teaspoons, and US cups.

## ✨ Key Features

- **Live Updates:** Results update instantly as you type.
- **Reusable Components:** Modular architecture using a shared `ResultatCard` for a consistent UI/UX.
- **Smart Dropdowns:** Easy navigation with flags and clear unit labels.
- **Custom Design:** Unique background images and color schemes tailored for each category.
- **Error Handling:** Robust logic to handle empty inputs and API edge cases.

## 🛠 Tech Stack
- TypeScript 

- [React Native](https://reactnative.dev/) & [Expo](https://expo.dev/)
- [React Native Element Dropdown](https://github.com/hoaphantn7604/react-native-element-dropdown)
- [Frankfurter API](https://www.frankfurter.app/) for reliable currency data.
